# florence by mills — Brand Website

顶级品牌展示网站，为 Florence by Mills（Millie Bobby Brown 创立的纯素零残忍美妆品牌）打造。

## 预览

直接用浏览器打开 `index.html`。

## 技术栈

- 纯 HTML/CSS/JavaScript，零依赖
- Playfair Display + Outfit 字体（Google Fonts）
- IntersectionObserver 滚动动画
- SVG 波浪分割线 + 毛玻璃卡片
- CSS 星空粒子 + 品牌跑马灯
- Tweaks 面板（右下角齿轮按钮）
- 完全响应式

## 设计系统

| Token | 值 |
|---|---|
| Purple | `#7069BC` |
| Teal | `#0E7A82` |
| Navy | `#272D45` |
| Cream | `#FDFBF7` |
| Heading | Playfair Display |
| Body | Outfit |
