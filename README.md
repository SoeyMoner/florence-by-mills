# florence by mills — Brand Website

顶级品牌展示网站，为 Florence by Mills（Millie Bobby Brown 创立的纯素零残忍美妆品牌）打造。

## 预览

直接用浏览器打开 `index.html`：

```
start index.html
```

## 技术栈

- 纯 HTML/CSS/JavaScript，零依赖
- Playfair Display + Outfit 字体（Google Fonts）
- IntersectionObserver 滚动动画
- SVG 装饰元素（波浪分割线、图标）
- CSS 星空粒子、品牌跑马灯
- 毛玻璃卡片效果
- 响应式设计（桌面 + 平板 + 手机）
- Tweaks 面板（右下角齿轮按钮）：主题色 / 字号 / 暗色模式 / 动画开关

## 设计系统

| Token | 值 |
|---|---|
| 主色 Purple | `#7069BC` |
| 辅色 Teal | `#0E7A82` |
| 深色 Navy | `#272D45` |
| 背景 Cream | `#FDFBF7` |
| 标题字体 | Playfair Display |
| 正文字体 | Outfit |

## 图片资源

`img/` 目录包含：
- 13 张 Millie Bobby Brown Instagram 照片
- 12 张官网产品图

`public/` 目录包含品牌营销素材。
